# Python Coding Projects

This repository contains Python practice projects and exercises inspired by the book *Coding Projects in Python* published by Penguin Random House.

The purpose of this repository is to practice Python programming fundamentals by building small projects and experimenting with core concepts.

## Topics Practiced

- Variables and data types
- Conditional statements
- Loops
- Functions
- Modules
- Basic problem solving

Each project helps reinforce programming concepts through hands-on coding.

## Project Structure

```
python-coding-projects/
│
├── project_01/
├── project_02/
├── project_03/
└── README.md
```

Each folder contains a separate Python project or exercise.

## Requirements

- Python 3.x
- Any code editor (VS Code, PyCharm, etc.)

## Running the Projects

Clone the repository:

```
git clone https://github.com/yourusername/python-coding-projects.git
cd python-coding-projects
```

Run a project:

```
python project_01/main.py
```
